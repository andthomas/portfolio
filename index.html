TicTacToe

TicTacToe was my first independent development project. The game is built entirely with HTML, CSS, Javascript and JQuery. All animations where produced using JQuery animate functions. Local storage was used to store the player's scores between browser refreshes and closes. An AI component was incorporated to allow the player to play against the computer. Otherwise, the default mode is for 2 players. The game sores and game state can be entirely reset by clicking the reset button at the bottom of the screen. (Source code)

Happy Hour

Happy Hour is a social media app allowing bar owners to post their happy hour times and deals to a venue profile. Users can then browse the profiles in a list view or on an embedded Google Map to see what deals are available at their local venue. The back-end of the project is built with Ruby on Rails. A CRUD system allows for users, venues, happy hour times and deals to be created and associated to the correct venue. Users have logins and can upload images of their bar to the site (hosted on Cloudinary) and a geocoder takes an address and plots it on a Google Map. (Source code)

Voyager 1

In this project, I constructed a 3D model of the solar system using Three.js. Integrated into this are a Blender model of Voyager 1 and its historical heliocentric trajectory data, both sourced from NASA's Space Physics Data Facility. These were converted into JSON format to render the model and map its flight trajectory. The model satellite is animated to move through the solar system which can be controlled by a Dat GUI controller. The camera angle of the satellite can be manoeuvred using a Google Daydream controller. (Source code)
